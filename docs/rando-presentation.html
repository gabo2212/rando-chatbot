02 / TECHBETTER-T-STACK

Under the hood

How it's built

npm monorepo with Turborepo. Next.js app talks to Postgres, OpenAI, and local Ollama for embeddings.

App layer

  • Next.js 16 App Router (port 3001)
  • React 19 + Tailwind 4
  • tRPC for chat CRUD
  • REST for AI stream, docs, escape, auth

Data & AI

  • Postgres 16 + pgvector (Docker)
  • Drizzle ORM · Better Auth
  • OpenAI for chat streaming
  • Ollama nomic-embed-text for RAG

07 / PLAYEMBRACE THE MADNESS

Extra surfaces

Games inside the protocol

THE LAST TERMINAL

In-chat escape room narrated by A.R.I.A. Deterministic engine, persisted runs, slash commands, and AI narration over a sealed facility puzzle.

ENDLESS DRIVE

Full-bleed iframe racer at /drive — procedural low-poly world, nitro, power-slide drift, night headlights, seed worlds (DRIVE-2049).

Built as a standalone Three.js (r185) WebGL page in public/games/, embedded via iframe on /drive. Chunked terrain from a string seed hash, custom sky shader, day/night cycle, Web Audio engine loops — no React inside the game loop.
